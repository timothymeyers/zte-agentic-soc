# zte-agentic-soc
Implementation of an Agentic Security Operations Center MVP

## Custom Agents

This repository includes custom GitHub Copilot agents to enhance development workflows:

### Python Developer Agent

Located at `.github/agents/python-developer.md`, this expert agent provides:

- **Python Best Practices**: PEP 8, PEP 257, SOLID principles, and modern Python idioms
- **Pytest Testing**: Comprehensive testing patterns including fixtures, parametrize, and mocking
- **MCP Server Integration**: Uses Context7 tools to research package documentation and examples
- **Code Quality**: Type hints, docstrings, error handling, and security considerations
- **Development Workflow**: From requirements analysis to code review

The Python Developer Agent helps maintain high-quality, well-tested Python code throughout the project.
